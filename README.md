# Barcode and SKU Auto Generator

Automatically generates Barcodes and SKUs for ERPNext v15 Items.

## Features
- Auto generate Barcodes and Custom SKUs on item creation.
- Define custom naming logic using the "Barcode SKU Settings" DocType.
- Mass re-generate barcodes/SKUs for existing Items that don't have them.

## Setup
```bash
bench get-app barcodesku
bench install-app barcodesku
```
